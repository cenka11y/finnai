## Implementation Notes

### Translation Strategy
1. **Fallback Chain**: `fi` → `en` → user's selected language
2. **Dynamic Loading**: Load only needed language packs
3. **Pluralization**: Use ICU message format for complex plurals
4. **Variables**: Support parameter interpolation `{variable}`
5. **Rich Text**: Support basic HTML tags in translations

### Key Conventions
- Use dot notation for nested keys: `navigation.dashboard`
- Placeholder format: `[LANG_CODE] content` for untranslated strings
- Variable format: `{variableName}` for dynamic content
- Keep keys semantic, not literal translations

### Cultural Considerations
- **Finnish**: Direct, formal tone; use "sinä" (informal you)
- **Arabic**: RTL layout support; formal register
- **Turkish**: Respectful tone; use "siz" (formal you) 
- **Russian**: Formal tone; Cyrillic script support
- **Somali**: Community-oriented language; inclusive tone

### Technical Requirements
- UTF-8 encoding for all character sets
- ICU message format for pluralization
- Gender-neutral language where possible
- Consistent terminology across languages
- Context notes for translators in separate file

### Quality Assurance
- Native speaker review for each language
- Cultural appropriateness testing
- Consistency checks across languages
- A/B testing for key conversion points
- Regular updates based on user feedback

This copy deck serves as the foundation for all user-facing text in SOUAI, ensuring consistent, culturally appropriate communication across all supported languages.
